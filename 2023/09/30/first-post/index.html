Leetcode Problem no. 1093

Problem Statement

Leetcode problem no. 1093 is about statistics from a large sample. A large sample of integers in the range [0,255]. It is represented by an array count where count[k] is the number of times that k appears in the sample.

Calculate the following statistics:

  • The minimum element, minimum in the sample.
  • The maximum element maximum in the sample.
  • The average of the sample, mean, calculated as the total sum of all elements divided by the total number of elements.
  • median:
    • If the sample has an odd number of elements, then the median is the middle element once the sample is sorted.
    • If the sample has an even number of elements, then the median is the average of the two middle elements once the sample is sorted.
  • mode: The number that appears the most in the sample. It is guaranteed to be unique.

Return the statistics of the sample as an array of floating-point numbers [minimum, maximum, mean, median, mode]. Answers within 10^{-5} of the actual answer will be accepted.

Solution Proposed

My idea is to use the data structure, queue, embedding another structure, hash, to take the input data, count. While constructing this data structure, dig_queue by iterating count, I can first compute a statistical quantity, mode, the value summed over count, and keep the value greater than zero in the queue.

Secondly, when I have dig_queue, I can easily know the minimum and maximum by looking at the front and the rear elements, respectively. Meanwhile, mean can be computed as well.

The last thing is to determine median. This can be done by checking the size of dig_queue to see whether the size is odd or even. Then median can be obtained accrodingly.

Conclusion

The proposed solution has the fastest runtime but the worse efficiency in memory use.
